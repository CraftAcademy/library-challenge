User's Story
1)
```
As a Library owner
In order to keep track of all my books 
I need an app that allows me to manage them
```
2)
```
As a Library owner
In order to be allow both the visitors and myself see the books we have available 
I need a list that sorts the books by their title author

```
3)
```
As a Library owner
In order to be able to check if a book is available or not
I need a function that allows me to traverse a file that contains a list of books 
```
4)
```
As a Library owner
In order to allow the visitors to check out the books they want 
I need a search function that allows finding the book by it's title or it's author
```
5)
```
As a Library owner 
In order to retrieve my books needs within a certain time (one month)
I need a function that lets the visitor know what date they should return the book
```
6)
```
As a Visitor
I need to have an account with the library 
The account has to allow me to check out book and return them
```
