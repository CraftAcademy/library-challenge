User Stories

As a Customer

I want to see a list of available books with the title and the author.
I want to know the date I need to return the book.

As the Library

I would like to store the books titles and authors in a application.
I would like to easily be able to see if the book is available or not.
I would like it so customers can checkout their books.
I would like it so customers can return their books.
I would like the books to have a return date of one month.
